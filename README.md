Enterprise Networking Lab

This repository contains my networking labs created using Cisco Packet Tracer.

The goal of these labs is to practice networking fundamentals such as LAN configuration, IP addressing, and router configuration.

--------------------------------------------------

Day 1 — Basic LAN Network

In this lab I created a simple network with two PCs connected to a switch.

Steps I performed:

1. Added PC0, PC1 and a 2960 switch.
2. Connected both PCs to the switch using copper straight-through cables.
3. Assigned IP addresses to each device.

PC0: 192.168.1.10
PC1: 192.168.1.11

4. Used the ping command to test communication.

The ping test was successful which confirmed both devices were able to communicate across the network.

What I learned:

• A switch connects devices inside the same network  
• Devices require unique IP addresses  
• Ping is used to test connectivity between devices
